CSV Format

The CSV file should have the following columns:

Column Required Description
name Yes Company name
gstin No GSTIN number (e.g., 27AAAAA0000A1Z5)
pan No PAN number (e.g., AAAAA0000A)
address No Company address
is_default No Set as default (1 or true). Only one company can be default.

Example:

name,gstin,pan,address,is_default
Acme Pvt Ltd,27AAAAA0000A1Z5,AAAAA0000A,"123 Main St, Mumbai",1
Tech Corp,27BBBBB0000B1Z5,BBBBB0000B,"456 Tech Park, Bangalore",0
